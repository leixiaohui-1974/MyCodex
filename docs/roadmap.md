# MyCodex Roadmap

## Phase 1: MVP Foundation (current)

- desktop shell
- project management basics
- team launcher integration
- task and goal execution flow
- backup and auth entry points

## Phase 2: Result Experience

- artifact browser (summary, judge, merge outputs)
- markdown/html preview panel
- run timeline and lane status cards

## Phase 3: Account and Security

- full Google OAuth callback and token lifecycle
- secure credential storage
- per-project permission policies

## Phase 4: Productivity and Stability

- retry and resume controls from UI
- team presets per task type
- observability and diagnostics
- packaging and release automation
